## 📱 产品简介

**PhotoVault Pro** 是一款基于 Face ID 的照片加密存储应用，帮助用户保护隐私照片安全。

### 核心特色

- 🔒 **Face ID 锁定** - 生物识别，安全可靠
- 📁 **隐私相册** - 隐藏照片，独立密码
- 🎨 **伪装模式** - 伪装成计算器/其他工具
- 📤 **安全导入** - 从系统相册安全导入
- 🌙 **深色模式** - 支持深色主题
